VisionFlow: Cloud-Native Image Processing Pipeline
A-Z DevOps project demonstrating Microservices, IaC, CI/CD, and Kubernetes.
🛠 Tech Stack
App: FastAPI (Python), Redis, PostgreSQL
Containerization: Docker & Docker Compose
Infrastructure as Code: Terraform & LocalStack (AWS S3 simulation)
CI/CD: GitHub Actions (Automated Linting & Security Scans)
Orchestration: Kubernetes (Kind Cluster)
Monitoring: Prometheus & Grafana (Installed via Helm)
🚀 Key Features
Asynchronous Processing: API decoupled from Worker via Redis.
Scalability: Horizontal Pod Autoscaling in Kubernetes.
Security: DevSecOps pipeline with automated checks.
Observability: Real-time Grafana dashboards for cluster health.
